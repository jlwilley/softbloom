# Logo assets

Once the logo is finalized, drop these files here:

- `jw-primary.svg` — serif monogram with petal accent (primary mark)
- `jw-mono.svg` — monochrome variant (no petal, or petal in same color)
- `jw-primary-dark.svg` — optimized for dark backgrounds (if needed)
- `petal.svg` — standalone five-petal bloom (secondary mark, app icons)
- `wordmark.svg` — "jaredw" wordmark for horizontal lockups
- `favicon/favicon.svg` — SVG favicon
- `favicon/favicon-16.png`
- `favicon/favicon-32.png`
- `favicon/apple-touch-icon.png` (180x180)
- `favicon/favicon.ico`

## Design constraints

- All SVGs should use `currentColor` for fills where possible, so they respond to CSS color.
- Monogram mark must work at 16x16 (favicon) through 512x512 (hero).
- Petal geometry: five ellipses rotated 72° around a center point, small center circle.
- Primary color: `--sakura-900` (#5C1A2E) or `--sakura-600` (#B84668).
- Accent petal color: `--sakura-400` (#E87FA0) or `--sakura-200` (#F5B8CD).

## Clear space

Minimum clear space around the primary mark: equal to the height of the "w" character.

## Minimum size

Primary mark: 24px height minimum.
Petal-only mark: 16px minimum.
